
##How to add courses to the list

All courses should be added in separate brunch, before merging with main to avoid crashes.
To add a new course, you should open /courses.json and go to the bottom of the file.
After that you can add a new course.

id - is an identification number of the course.
title - title of the course.
dimension - is it course 3D or 2D, you need to pick the dimension.
category - every course divided by different categories
level - every course has a level from beginner to advanced
thumbnail - is a head image of the course. Before uploading it to https://ibb.co/ or any other image hosting please convert it to JPG format and resize. Image or GIF should be under 1.2MB of size.
screenshots - every course has 2 screenshots. thumbnail rule of resizing and converting applies here as well.
tags - to make searching easier we add tags to the course
hours - the lenght of video materal of the course
description - description of the course
software - software used in the course
language - language of the course
subtitles - does the course has subtitles
materials - does materials included in the course
downloadLink - link to the download
sourceLink - link to the source
